# FileHash

Use this tool for generate a filehash value of a file.

You could drag and drop a file in the app, then click the Compute button to generate the filehash value.
The tool can create a file with the filehash value inside.

**Build requirements**:
* Visual Studio 2022
* .Net Framework 3.5 development tools

**Use requirements**:
* .Net Framework 3.5 and upper
